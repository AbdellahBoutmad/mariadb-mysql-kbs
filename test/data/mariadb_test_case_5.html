tokudb_version

  • Description: The TokuDB version of the plugin included on MariaDB.
  • Dynamic: No
  • Data Type: string

tokudb_write_status_frequency

  • Description: Progress is measured every this many writes for display by SHOW PROCESSLIST. Useful to set to 1 to examine slow queries.
  • Scope: Global,
  • Dynamic: Yes
  • Data Type: numeric
  • Default Value: 1000
  • Range: 0 to 4294967295
